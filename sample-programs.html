Team Feed Sports App

This sports information app was a Rutgers Coding Bootcamp group assignment. It helps novices contribute to conversations & keep fans up tospeed. Built with HTML, CSS, Google Fonts, Bootstrap, Javascript, jQuery ,Ajax, Json: Api keys for Wikipedia, Giphy, Facebook, Stattleship, & Firebase.

Force Off Game

This Star Wars inspired fight game was an assignment for Rutgers Coding Bootcamp. Users have can choose 1 of 4 characters, each with a different strength & attack power. After choosing a character, users "fight" the remaining characters: scores are displayed under the fighters. Built with HTML, CSS, Javascript, & jQuery.

WWI Hangman

History themed hangman game. Users have to guess the name of World War One combatant nations 1 letter at a time. They are allowed to make 9 mistakes before losing the game. Uses HTML, Bootstrap,& Javascript

Distopian Trivia Game

A timed, multiple choice trivia game. After clicking the start button, users have 150 seconds to match clues to 1 of 4 potential answers. Built as a Rutgers Coding Bootcamp assignment. Uses HTML, Bootstrap, Javascript, & Ajax.
