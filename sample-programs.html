Team Feed Sports App
This was a group project at Rutgers Coding
Bootcamp. It helps non sports fans contribute
to conversations and keeps fans up to speed
by quickly searching for a teams news, social
feed, & roster. Built with HTML, CSS,
Bootstrap, Javascript, jQuery ,Ajax, Json.
Firebase stores user input. Api keys for
Wikipedia, Giphy, Facebook, Stattleship
retrieve data.
    

Force Off Game
This Star Wars inspired fight game was an
assignment for Rutgers Coding Bootcamp.
Users have can choose 1 of 4 characters, each
with a different strength & attack power. After
choosing their character, users "fight" the remaining character: their respective scores are displayed underneath their fight box positions. Built with HTML, CSS, Javascript, & jQuery.
