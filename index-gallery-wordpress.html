These are some the apps I worked on while attending Rutgers Coding Bootcamp.
FreeSpace, Resume Padder, & Team Feed were group projects, the rest I built alone. Try them out by clicking of the photo or description.

A crowd sourced, mobile app that allows users to share the location of open parking spaces. Uses React Native, Javascript, Node.js, Google Map API, Geolocation API, Native Base API, MongoDB.

A Shark themed role player game. Users choose 1 of 4 sharks, each has different strengths & attack powers. Sharks "fight" each other: scores are displayed under the fighters. Uses HTML, CSS, Javascript, & jQuery.

History themed hangman game. Users guess the name of World War One combatant nations 1 letter at a time & allowed 9 mistakes before losing the game. Uses HTML, Bootstrap,& Javascript

A multiple choice trivia game inspired by various distopian fiction: with an equally sterile color theme. Users have 150 seconds to match clues to 1 of 4 answers. Uses HTML, Bootstrap, Javascript , & Ajax.

A train schedule app that takes user input for destination, train name, frequency, & eta. Uses HTML, Bootstrap,
Javascript, Firebase ,& Ajax.













A simple bubble sort algorithm. Uses HTML, CSS, & Javascript

A calculator that performs basic math. Uses HTML, BootStrap, Javascript, & jQuery.

A digital stop watch. Allows user to start, stop, reset, & mark laps. Uses HTML, CSS, Javascript, & jQuery

View public information about sports teams social feeds, events, & gifs. Uses HTML, CSS, Javascript, Bootstrap, AJAX, jQuery, Google fonts, ESPN API, Giphy API, Facebook API, Stattlefish API, Wikipedia Opensearch API, jQuery Autocomplete UI, Firebase.

A site for students to find part-time jobs, volunteer work, & internship opportunities. Uses HTML, CSS, Bootstrap, Javascript, jQuery, Node.js, Express.js, Handlebars.js, Sequelize.js,Bcrypt.js, Nodemailer.
